Supporting Projects

DockerX

Julia interface to the Docker API for managing containers and gathering metrics. This package is based on the original Docker package, but updated to serve our own needs. I've also added CI to the build process.

PAPI

Updated bindings to the PAPI Library, forked from the original PAPI.jl which has not been updated in 4 years. This package includes the PAPI executable for reproducibility, courtesy of PAPIBuilder2, which is auto built with BinaryBuilder.

In essence, this gives access to a processor's hardware event counters through perf.

PAPIBuilder2

Builder for the PAPI library. Releases fetched when installing can be found here: https://github.com/hildebrandmw/PAPIBuilder2/releases

MemSnoop

Snooping routines to gather metrics on running programs. Includes the following analyses:

  • Idle page tracking
  • Hardware performance counter tracking through PAPI.

One of the big goals of this package is to reduce the number of third party dependencies as much as possible since Idle Page Tracking requires Julia to be run as root.
