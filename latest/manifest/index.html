Supporting Projects

Docker

Julia interface to the Docker API for managing containers and gathering metrics. This package is based on the original Docker package, but updated to serve our own needs. I've also added CI to the build process.

PAPI

Updated bindings to the PAPI Library, forked from the original PAPI.jl which has not been updated in 4 years. This package includes the PAPI executable for reproducibility, courtesy of PAPIBuilder2, which is auto built with BinaryBuilder.

In essence, this gives access to a processor's hardware event counters through perf.

PAPIBuilder2

Builder for the PAPI library. Releases fetched when installing can be found here: https://github.com/hildebrandmw/PAPIBuilder2/releases

SystemSnoop

Snooping routines to gather metrics on running programs. Includes the following analyses:

  • Idle page tracking
  • Hardware performance counter tracking through PAPI.

One of the big goals of this package is to reduce the number of third party dependencies as much as possible since Idle Page Tracking requires Julia to be run as root.

ml-notebooks (private)

Jupyter notebooks and scripts for research.

X-Mem

Fork of the Microsoft X-Mem repositoy. I've added a little bit to it to

  • Ensure latency pointer chasing traverses the whole working set.

  • Allows a file to be used the source for memory allocation of the working set. This lets us a memory mapped file on a persistent memory device to give us direct access to that device so we can take measurements of it, assuming the file is mounted on a direct access file system. (worst sentence ever)

XMem

Julia package for dealing with X-Mem. Auto builds the binary, launches it, controls flags, all that fun stuff.

Checkpoints

I got very tired of suffering jupyter notebook apocalypses. This pacakge helps avoid that by managing storing results of long running computations so I can close down the notebook but still have the results. It's actuallyh quite convenient.

PCM

Very experimental wrapper around PCM, but provides access to per-channel DRAM read/write counters and per-dimm PMEM read/write counters, as well as DRAM cache hit-rate when operating in 2LM.
